# C Programming Practice

This repository contains various C programs I have written to practice and understand the basics of C language and Data Structures.

## 📁 File Descriptions

- `basic.c` – Basic C programs (input/output, loops, conditionals)
- `conversion.c` – Programs related to number or unit conversions
- `evaluation.c` – Expression evaluation (like postfix, infix, etc.)
- `file.c` – File handling operations in C
- `linkedlist.c` – Implementation of Linked Lists
- `search.c` – Searching algorithms (linear, binary)
- `sorting.c` – Sorting algorithms (bubble, selection, etc.)
- `stack.c` – Stack implementation using arrays or linked lists
- `zak.txt` – Notes or sample input/output
